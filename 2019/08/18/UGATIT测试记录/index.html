3. 项目需要用的资源

原项目地址(tensorflow 版本)：https://github.com/taki0112/UGATIT

python(官网)：https://www.python.org/

tensorflow(pip 命令安装):pip install tensorflow

open-cv(pip 命令安装):pip install open opencv-python

如果依赖中有某些没有安装，寄几个搜吧。（也可以直接问我）

4. 过程中遇到的问题

主要是预训练模型无处可寻，最终找到两个来源。

  • 大佬自己训练的，解压后大小约 1.5G。风格偏向彩铅，边界不太清晰，实际效果五六张里能有一张好看的吧。

  • 官方版本，解压后大小约 7.5G。风格偏向油画，边界较清晰，实际效果三张中有一张较好的。

    官方版本解压的时候会出现问题，7z 文件我也不知道怎么办…

对于 zip 文件，先检查压缩包的 md5 校验值与下载的 md5 文件数据是否相同，如果相同则数据文件内容没有损坏，可以在 macos 上解压。（我专门安装了一个 mac 的虚拟机…)(如果需要的同学可以联系我)

使用方法教程：https://zhuanlan.zhihu.com/p/76979105
